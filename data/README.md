# Data Overview

This folder holds simple structured files that can support dashboards later.

The first goal is not sophistication. The goal is consistency.

## Current files

- `weekly-scorecard.csv`: one row per ministry week
- `follow-up-pipeline.csv`: one row per follow-up item or person response

## Suggested use

- Update these files weekly or export comparable data from your ministry systems
- Use them to build simple reports in a spreadsheet, BI tool, or future app
- Keep field names stable so reporting becomes easier over time
