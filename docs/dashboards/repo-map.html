Markdown in Chrome

How hard is nicer reading?

Chrome by itself

Easy but plain. Chrome can open `.md` files from disk, but it renders them as raw text. This is fine for quick inspection and not great for reading.

Browser extension

Easiest personal option. Install a Markdown viewer extension and allow file URL access. Low effort, but it lives in your Chrome setup rather than in the repo.

Local reader page

Medium-light. Add a `reader.html` that fetches Markdown and renders it in-browser. Chrome usually needs a tiny local server for this, because `file://` fetches are restricted.

Generated mini-site

Most polished. Generate HTML from Markdown into a browsable mini-wiki. Still lightweight, but it adds a build step. Worth it only if this map starts getting real daily use.
