# Dashboards Overview

Dashboards should eventually make it easy to answer a few simple questions:

- Are we healthy?
- Are people being followed up with?
- Are weekly responsibilities getting completed?
- Where is the online campus pastor overloaded?

## Recommended dashboard types

- Weekly ministry scorecard
- Connection pipeline dashboard
- Serve team coverage dashboard
- Operational readiness dashboard
