# Role Map

The online campus pastor role currently appears to combine several distinct hats. Naming them is the first step toward documentation, delegation, and healthy staffing.

## Strategy and leadership hats

- Vision owner
- Ministry architect
- Metrics reviewer
- Team builder

## Operations hats

- Weekend producer
- Script writer
- Service coordinator
- Host scheduler
- Giveaway coordinator
- Admin handoff owner
- Analytics reporter
- Systems documenter

## Technical hats

- Streaming operator or stream owner
- Studio coordinator
- Troubleshooter
- Platform administrator

## People and pastoral hats

- Online pastor
- Follow-up owner
- Prayer and care responder
- Connection pipeline manager
- Key leader care owner
- Community group coach

## What this role map is for

Use this document to decide:

- Which hats truly belong to the pastor role
- Which hats should become staff roles
- Which hats can become serve team roles
- Which hats need documentation before they can be delegated
