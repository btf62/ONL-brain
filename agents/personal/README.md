# Personal Agents

This folder contains agents that support the person carrying the work, not only the work itself.

These agents may help with:

- prioritization
- task breakdown
- overwhelm reduction
- reflection
- personal operating rhythms
- emotionally grounded execution

## Current agents

- [CBT Coach](cbt-coach.md)

## Boundary

These agents can be deeply helpful, but they should still be treated as support systems rather than licensed clinicians, pastors, or final authorities.
